THE LEARN TO CODE WORKSHOP

  • Ruby on Rails is the fastest growing web development framework today. This course will teach you all you need to know to get your first web app up and running. Ruby on Rails has proved itself as versatile platform to use from nimble startups through to the established companies.
    The workshop is designed to give you entry level skills to developing web apps or to learn the tech talk needed to deal with developers for your startup.

    Be a part of the future. Learn to code.

WHAT CAN I LEARN?

  • The developer's toolkit:
  • Set up your 'development environment' and learn about open source tools
  • Programming fundamentals:
  • Learn the 5 fundamentals of computer programming
  • Application and database design:
  • Understanding these processes helps you build great apps
  • Ruby on Rails:
  • Build a web application that you'll be proud to show your friends

WHAT CAN I EXPECT?

  • As a class, we will set up your development environemt as this is key to setting up, testing and deploying new apps. We will teach you the skills to create new apps, and ways to speed up deployment using scaffolding.
    This course will give you an understanding of Ruby gems that enable greater functionality for your app, and give you the skills needed to implement popular gems into your apps.
    The course is a fun and intereactive way of learning the fundamentals behind coding in Ruby on Rails

THE DETAILS

  • Times: 10am - 4pm.

  • What to bring:
  • Your own laptop, lunch money.

  • We promise:
  • Small groups of less than 12.
  • An better understanding of Ruby on Rails.
