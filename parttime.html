THE WEB APP DEVELOPER COURSE

  • Coder Factory's part time web app developer course allows you to develop the skills needed to build web applications whilst keeping your day job. Our 12 week evening or weekend course teaches you full stack web application development using the latest technologies.

    Online tutorials will only take you so far. Whether you are looking to build your own tech startup, enhance your career prospects or just learn more about web apps, this programme will get you there.

WHAT CAN I LEARN?

  • Web design using HTML, CSS and Bootstrap
  • Database design
  • Application design
  • Ruby and object-oriented programming
  • Ruby on Rails - Rapid Application Development
  • Leveraging the power of Ruby gems
  • Collaboration using GitHub
  • Deploying your app to Heroku
  • Web optimisation
  • ... and much more

WHAT CAN I EXPECT?

  • You'll learn in a small class environment and collaborate with other students to produce a social networking web app. You'll also have the opportunity to build your own web app from day 1, with help and guidance from our teachers. We build our apps using Ruby on Rails and we'll make them look beautiful using HTML5 and CSS3.

    You will meet great poeple, build your network and most of all, have a lot of fun!
