CODING FOR HIGHSCHOOLERS WORKSHOP

  • Coding is being taught at highschools all across the country. Younger kids start on Scratch in primary school, and develop their skills throughout their schooling career. Knowing how to code can help you understand technology better, give you more options when choosing your career, and could even start you earning money before you leave school.
    This workshop is designed for high schoolers between the ages of 13 and 17. It will teach you more advanced coding skills using Ruby on Rails.
    Be a part of the future. Learn to code.

WHAT CAN I LEARN?

  • The tools of the trade:
  • Set up your 'dev environment' and learn what free software is available
  • Programming
  • Learn the 5 fundamentals of computer programming
  • Web application development
  • Learn how easy it is to deploy an app live on the web
  • Ruby on Rails - the best web application framework
  • Build an Instagram styled web application that you and your mates will be able to use.

WHAT CAN I EXPECT?

  • You'll learn in a small class environment and collaborate with other students to build an Instagram styled social media app.
    We will set up your development environment and teach you the basics to enable you to deploy your first app.

    You'll have the opportunity to ask lots of questions, and the teachers will be able to guide you and help develop your coding skills as you build your app using Ruby on Rails.

THE DETAILS

  • Days: Saturday & Sunday
  • Times: 10am - 4pm

  • What to bring:
  • Your own laptop, lunch money

  • We Promise:
  • Small groups of less than 12
  • An understanding of web apps in simple terms
