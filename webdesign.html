THE WEB DESIGN WORKSHOP

  • Great web design is your differentiator. This course will give you a solid understanding of HTML and CSS which can be used on apps built in Ruby on Rails, Wordpress or other web based technologies.

    This course will give you the skills to allow you to build websites that stand out from the crowd. HTML and CSS are the building blocks of the web and a great entry point into learning more about web design.

WHAT CAN I LEARN?

  • An overview of the key web design technolgies
  • The inner workings of the web
  • How to design and plan a website
  • Using wireframes for prototyping
  • HTML5 and CSS3
  • Kickstarting projects using Bootstrap
  • Setting up your hosting your website
  • Optimising your site for SEO
  • Using Google Analytics to improve your site

WHAT CAN I EXPECT?

  • As a class, we will take a website all the way from mock-ups and prototyping, to deploying and hosting it on the web. You will also have the opportunity to build your own website from scratch with the help and guidance of our teachers.

    We will teach you the latest tips and tricks and show you where to find a stack of great free tools to help you build your website quickly and efficiently.

THE DETAILS

  • Days: Saturday & Sunday
  • Times: 10am - 4pm.

  • What to bring:
  • Your own laptop, lunch money.

  • Small groups of less than 12.
  • An understanding of web design in simple terms.
